Nucleus Plugin Popup Help

Template Name

A template name consists of only the characters a-z and 0-9, and cannot contain spaces.

Blog List Template

Variables that can be used in all parts

Template Var. Description
<%skinfile%> URL of skin directory. You can use <%skinfile(filename)%> too.

Variables that can be used in blog list item

Template Var. Description
<%blogid%> Blog ID
<%bloglink%> Link of blog (only URL).
<%archivedate%> Archivedate (in Archive skin)
<%blogurl%> Blog URL set on "Blog Settings" page of Nucleus. If empty, it is replaced by default blog URL or $CONF['Self'].
<%blogname%> Blog name
<%bshortname%> Blog short name
<%blogdesc%> Blog discription
<%flag%> The content of "Blog List Flag" is displayed in this part of current selected blog.
<%self%> Variable of $CONF['Self']
<%%categorylist%%> Category list of the blog. (If the module is enabled.) Use "%%" insted of "%".
<%%PLUGIN NAME%%> BlogMenu calls another plugin.
e.g. <%%PLUGIN NAME(OPTION1,OPTION2)%%>
Template variable of BlogMenu can be used for the option value.
<%%+PLUGIN NAME%%> When "+" is put on the head, BlogMenu always calls the plugin.

Variables that can be used in blog list flag

Same variables that can be used in blog list items except <%flag%> and other plugins.

Blog Aliases

BlogMenu prints the alias of a blog insted of the name. Put a line break between blogs. 

blogid=aliase
e.g. 
1=Home
5=Apendix

Variables that can be used in category list item

<%catid%> Category ID
<%catlink%> Category link (only URL)
<%catname%> Category name
<%catdesc%> Category description
<%amount%> Item amount of the category
<%catflag%> The content of "Category List Flag" is displayed in this part of current selected category.

Variables that can be used in category list flag

Same variables that can be used in category list items except <%catflag%> and other plugins.

Display priority

The order of the display can be permuted. It's start from 1.

Module

It is possible to call it from the template as a module by putting the file of the PHP class according to the prescribed form on blogmenu/modules directory. However, it is necessary to make it to "Enable" first.

Basic settings of display priority

Default Priority Level

New blog/category will be set to this priority.

Maximum Priority Level

Number of levels that can be chosen by the priority select list. (1 - 999)
